### 💡 遇到的问题与备忘 (Issues & Notes)

- **Qlib Windows 安装问题**: 
  - ❌ `pip install pyqlib` 失败：PyPI 上不存在此包名
  - ❌ `pip install qlib` 安装的是错误的包（Q Engineering 的 qlib，不是 Microsoft 的）
  - ✅ **正确安装方式**：`pip install git+https://github.com/microsoft/qlib.git`
  - ⚠️ **Windows 编译依赖**：需要 Microsoft Visual C++ Build Tools（14.0+）
  - 📝 **解决方案**：
    1. 安装 Visual C++ Build Tools：https://visualstudio.microsoft.com/visual-cpp-build-tools/
    2. 从 GitHub 安装：`pip install git+https://github.com/microsoft/qlib.git`
    3. 或使用降级方案：系统会自动从 TimescaleDB 加载数据，无需 Qlib
- **文档更新**:
  - ✅ 创建了 `qlib-setup-guide.md` 完整安装指南
  - ✅ 更新了 `qlib-integration.md` 安装说明
  - ✅ 更新了 `pyproject.toml`，注释掉 Qlib 依赖（避免自动安装失败）
- **Linter 警告**: 由于 Qlib 尚未安装，IDE 会显示导入警告，这是正常的。实际运行时需要先安装 Qlib
- **数据格式**: Qlib 需要特定的数据格式（MultiIndex, 标准字段名），已实现自动转换
- **降级方案**: 实现了降级机制，当 Qlib 不可用时自动从数据库加载，保证系统可用性
- **实验管理**: Qlib 使用实验（Experiment）和记录器（Recorder）管理训练结果，已封装在 API 中
- **配置灵活性**: 支持通过环境变量配置 Qlib 相关参数，便于不同环境部署
