### 💡 遇到的问题与备忘 (Issues & Notes)
- **ProcessBuilder 缺陷**: 脚本启动慢、无法复用内存数据 (如 Pandas 大对象)、错误处理困难。升级为 HTTP 微服务架构完美解决了这些问题。
- **Spring Cloud 版本**: 注意 Spring Boot 2.7.x 需搭配 Spring Cloud 2021.0.x 版本。
- **FeignClient 注入失败**: 启动报错 `required a bean of type AiEngineClient`. 原因: 主启动类缺少 `@EnableFeignClients` 注解。修复: 已在 `App.java` 中添加该注解。
